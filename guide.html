Guide for playing this game...

(1. Game ko khelne ke liye aapko bs start button pr click krna h. Start button pr click krte hi ek confirm box open hoga jisme game khelne ki basics hogi agr aap game khelna chahte h to simply ok kre game start ho jayega.

(2. Game ko khelna bahot hi easy h bs apko game start hote hi bird ko samne se aa rhi walls se takrane se bachana h aur bird ko niche jane se bhi rkna h jiske liye aap fly button pr click kr ke bird ko uper ki or utha skte h.

(3. Bird ke walls se takrane ya niche gir jane pr game over ho jayega aur agr aap ise fir se khelna chaho to restart kr skte ho.
